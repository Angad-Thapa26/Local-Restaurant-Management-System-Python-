Local Restaurant Management System (Python)

A Python-based Restaurant Management System designed to run on a local network where one device acts as the server and other devices connect as clients.

This system allows restaurants to manage menu inventory, orders, tables, and order status in real time within a single Wi-Fi network, making it suitable for small to medium restaurants without requiring a cloud server.

The server machine (for example, a laptop at the reception or kitchen) hosts the application, while other devices such as waiter phones, tablets, or additional computers connect through the same network.

📌 Key Concept

The system follows a local server–client architecture.

Server Device
Runs the Python application
Manages menu, inventory, and order status
Displays system dashboard
Client Devices
Connect via the same Wi-Fi network
Access the system using the server's local address
Can create and manage orders

A QR code is generated by the server to simplify connection for waiter devices.

⚙️ Features
Restaurant Management
Add, edit, or remove menu items
Manage prices and inventory
Monitor restaurant operations from a central dashboard
Order Management

Orders can be tracked through different stages:

Order Created
Preparing
Ready
Delivered
Paid

This helps both kitchen and front desk staff monitor order flow efficiently.

Table Management
Assign orders to specific tables
Waiters can add orders directly from table selection
Add special instructions or notes for the kitchen
Multi-Device Access

Multiple devices connected to the same network can:

Create orders
Modify orders
View order status
Update delivery/payment status
QR Code Connection

The server displays a QR code that allows waiter devices to instantly connect to the system through the local network.5. The system will display:
    * A local server address
    * A QR code for device connection



📱 Connecting Client Devices

1. Connect the device to the same Wi-Fi network.
2. Scan the QR code displayed on the server.
3. Open the provided local address in the browser.
4. Start creating and managing orders.
